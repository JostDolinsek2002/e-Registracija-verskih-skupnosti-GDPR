# e-Registracija verskih skupnosti – GDPR nadgradnja

Projekt predstavlja simulacijo spremljanja nadgradnje digitalne storitve
»e-Registracija verskih skupnosti« z vidika varstva osebnih podatkov (GDPR).

Osnova projekta je idejna in prototipna zasnova aplikacije, pripravljena v
prejšnji študijski nalogi. V tem projektu je poudarek na upravljanju sprememb
v skladu s ciljem COBIT 2019 BAI06 – Upravljane IT spremembe.

Projekt uporablja funkcionalnosti GitHub (issues, project, labels, milestones
in wiki) za prikaz procesa obravnave, implementacije in zaključka sprememb.
